LibreChat Onboarding Tour Test

Open this file in your browser, then follow these steps:

  1. Open LibreChat at http://localhost:3090
  2. Sign in if needed
  3. Click the Settings gear icon in the navigation
  4. Look for the "Platform Tour" or "Start Tour" button in the General tab
  5. Click the button to start the tour

Expected Tour Steps:

  1. Welcome to SovereignAI! Let me show you around the platform.
  2. Choose from 6 powerful AI tools in the sidebar. Each tool has unique capabilities.
  3. This is where you interact with your AI assistant. Type your questions or commands here.
  4. Your conversations are saved here. You can revisit any previous session.
  5. Switch between tools here. Each tool maintains its own conversation context.
  6. You're all set! Select a tool and start your first conversation.

Troubleshooting:

  • If the tour button is not visible, check the browser console for errors
  • If clicking the button does nothing, look for JavaScript errors
  • If the tour starts but some steps fail, note which elements are missing
